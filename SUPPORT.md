# サポート — SANBA

## 質問・使い方の相談

まず [`README.md`](README.md) / [`docs/local-dev.md`](docs/local-dev.md) / [`docs/architecture.md`](docs/architecture.md) を確認してください。
解決しない場合は **[Issue](.github/ISSUE_TEMPLATE/)** を立ててください（bug / feature / config のテンプレートから選択）。

## バグ報告・機能要望

[Issue](https://github.com/godhuu0505/sanba/issues/new/choose) からお願いします。
Pull Request を送る場合は [`CONTRIBUTING.md`](CONTRIBUTING.md) を先に読んでください。

## 脆弱性の報告

**公開 Issue には書かないでください。** [`SECURITY.md`](SECURITY.md) の手順（GitHub Security Advisories）で非公開に報告してください。

## 対応時間について

本プロジェクトは個人運用のハッカソン提出物です。対応時間の保証はできませんが、Issue は確認次第対応します。
