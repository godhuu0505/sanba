# ADR-0034: アプリアイコン／ロゴを棒人間「サンバさん」に更新（ファビコン・Apple touch icon 整備）

- ステータス: Accepted（受理・2026-07-05）
- 日付: 2026-07-05
- 関連:
  - [ADR-0033](0033-ui-redesign-v2-sticker-paper.md)（UI 刷新 v2「白い紙の上の問答 — ステッカー×原色×動く棒人間」）＝**デザイン言語の正**。本 ADR はその**ロゴ／アプリアイコン意匠の差分**であり、トークン・情報設計・書体・a11y は不変。
  - [ADR-0025](0025-light-paper-redesign.md)（棒人間サンバさんの導入）＝系譜。

## コンテキスト

これまで `components/sanba/Logo.tsx` は「山吹の円章『産』」（一字明朝）を正とし、ADR-0033 でも
ロゴ円章はその意匠だった。一方で次の課題があった。

1. **アプリアイコン／ファビコンが未整備** — `apps/web/app/icon.*` が存在せず、ブラウザタブ・ホーム
   画面・PWA に SANBA のアイコンが出ていなかった。
2. **ブランドの主役が円章のみ** — ADR-0033 で「状態を伝える UI 部品」として棒人間サンバさんを
   確立したが、ロゴ／アイコンにはその人物像が反映されていなかった。

ステークホルダーが、棒人間サンバさん（胸に**山吹の産章**）を主役にしたアイコン意匠を確定した。
左に**問いの吹き出し**、右に**ひらめきの電球**を配し、産婆術＝「問いで相手の中から答えを引き出し、
要件が産まれる」を一枚で表す。

## 決定

ロゴ／アプリアイコンの意匠を **棒人間「サンバさん」** に更新する。トークン・情報設計・書体・a11y は
ADR-0033 のまま**不変**——変えるのはマーク意匠とアイコン資産のみ。

1. **アプリアイコン（詳細版）** — ステッカータイル（白面・墨枠）＋棒人間＋二連の問いの吹き出し
   （山吹＋白）＋ひらめきの電球。源泉 SVG は `apps/web/public/sanba-app-icon.svg`、ホーム画面用に
   `apps/web/app/apple-icon.png`（180px・紙背景）。Apple touch icon として Next App Router が自動配線。
2. **ファビコン（簡易版）** — 16–32px の視認性を優先し、`apps/web/app/icon.svg` は**棒人間＋産章のみ**
   （吹き出し・電球・詳細を省く）。Next App Router が `/icon.svg` を favicon として自動配線。
3. **ヘッダーロゴ** — `components/sanba/Logo.tsx` は棒人間の線画マーク＋ワードマーク "SANBA"。
   小サイズで潰れないよう吹き出し・電球・タイルは省き、色は既存トークン（`--sanba-frame` /
   `--sanba-gold` / `--sanba-surface`）を参照する。`size` / `wordmark` の API は維持し、旧 `glyph`
   prop（円章の字）は呼び出しが無いため廃止する。
4. **マーケ／ドキュメント用ロックアップ** — `apps/web/public/sanba-logo-lockup.svg`（マーク＋"SANBA"）。

**純黒は使わない・墨ベース**という ADR-0033 の原則を継承する。

## 影響

- 変更は `apps/web` のみ。`Logo` の公開 API（`size` / `wordmark`）は後方互換で、`AppHeader` など
  既存呼び出しはそのまま動く。banner 内 "SANBA" 表示を検証する既存テストも不変。
- 円章「産」の系譜は `Avatar` / `ChatBubble`（agent の既定グリフ「産」）に残り、失われない。
- 検証: `typecheck` / `lint` / `vitest`（27 件）/ `next build` を通過。ビルドのルートに `/icon.svg`・
  `/apple-icon.png` が生成され、favicon・Apple touch icon の自動配線を確認した。

## 却下した代替案

- **詳細版アイコンをそのままファビコンに流用**: 16px で吹き出し・電球が潰れて判読不能。簡易版を
  別途用意し、詳細版はアプリアイコン／ホーム画面に限定した。
- **円章「産」の踏襲**: 棒人間サンバさんを主役にするステークホルダー意向を優先。ただし産字は
  `Avatar` で継続し、系譜は保持する。
- **ラスタ画像（PNG）を正にする**: SVG を正とし、`apple-icon.png` は源泉 SVG から再生成可能に
  しておく（意匠変更の追従コストを下げる）。
