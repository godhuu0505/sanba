## 1. 体験フロー（マクロ一本道 ＋ 会話内タブ）

```
01 ホーム → 02 準備 →
  03 会話開始（開始前→接続→許可→失敗系）
    → 04 会話履歴 ⇄ 05 参考資料 ⇄ 06 要件絵巻   （= 同一シェルの3タブ・音声継続）
    → 07 判定（確定ゲート） → 08 結果（要件産婆結果）
```

- **マクロ**は一本道（戻る/終了のみ）。**会話フェーズ（04〜06）だけ**が 3 タブのシェル。
- 素材は**随時**投入でき、裏で解析が走り 05/06 に反映される（フローを止めない）。
- 判定は回答・アップロードのたびに**裏で更新**、終了押下で**判定画面**として提示。

## 2. 共通シェル — 固定 vs 可変

会話フェーズ（04 会話履歴 / 05 参考資料 / 06 要件絵巻）は**同一シェルのタブ違い**。

```
┌─────────────────────────────┐
│ ① ステータスバー                       │ 固定
│ ② ヘッダー（産・問答・● REC・時間・⏹終了）│ 固定
│ ③ ミニ状況（◆要件 ⚠未確定 📎資料）       │ 固定（値ライブ更新／タップで該当タブ）
│ ④ タブ（会話履歴 / 参考資料 / 要件絵巻）    │ 固定
│─────────────────────────────│
│   ⑤ 本文 ＝ タブで切替                  │ ★可変
│─────────────────────────────│
│ ⑥ 問い＋選択肢（最小⇄一覧⇄詳細・常時ピン）│ 固定（全タブ）
│ ⑦ ボトムバー 2行                       │ 固定
│    [消音][マイク] / [テキスト入力][▶]    │
└─────────────────────────────┘
```

### 常時表示（A）
| # | 要素 | 内容 |
|---|---|---|
| ① | ステータスバー | 時刻・電波 |
| ② | ヘッダー | 産アイコン（金彩）・「問答」・`● REC mm:ss`・経過タイマー・終了 `⏹` |
| ③ | ミニ状況 | `◆要件 N` / `⚠未確定 N`（深掘り対象）/ `📎資料 N（解析中…）`。タップで該当タブへ |
| ④ | タブ | 会話履歴 / 参考資料 / 要件絵巻（アクティブ＝金下線） |
| ⑥ | 問い＋選択肢 | 現在の問いを**全タブ常時ピン**。最小⇄一覧⇄詳細（§4） |
| ⑦ | ボトムバー | **2行固定**。1行目 `[消音][マイク]`／2行目 `[テキスト入力][▶送信]`（§5） |

### タブで切替（B・⑤中央のみ）
| タブ | 本文 |
|---|---|
| 会話履歴 | 吹き出し（産=左/自分=右）＋ partial 字幕（認識途中＝薄字） |
| 参考資料 | 投入済みファイル一覧（**解析進捗インライン**）＋「＋ 素材を追加」 |
| 要件絵巻 | MoSCoW 要件（確信度・出所・**閲覧のみ**）＋ 深掘り対象（未解消）リスト |

### シェルを使わない画面（専用レイアウト）
03 会話開始系 / 07 判定 / 08 結果（ヘッダーの一部のみ流用）。

## 4. 選択肢の 4 モード切替（最小 ⇄ 一覧 ⇄ 詳細 ⇄ 比較）

同一の「現在の問い＋選択肢」を**開示レベルで切り替える**。軸は**グラバー（ハンドル）**で
「上＝開く／下＝閉じる」、明示ボタンは同操作のショートカット。

```
最小構成（常時ピン）
  │  ⤢広げる / 上ドラッグ / 上スワイプ  ───▶  一覧（展開カード）
  │  chip 長押し（近道）─────────────────────────────────┐
  └────────────────────────────────────────────────┐  │
                                                     ▼  ▼
一覧 ──各行[詳細›]/行長押し──▶ 詳細確認（ボトムシート・暗幕） ⇄ 比較（効き目/留意で横並び）
  ◀⤡閉じる/下ドラッグ/下スワイプ─ 最小            ◀── ✕/下スワイプ/暗幕タップ ──
```

| モード | 役割 | 主要操作 |
|---|---|---|
| **最小構成** | ちら見＋即答。問い1行＋横スクロールchip＋`⤢広げる` | chipタップ＝回答／**chip長押し＝詳細（近道）**／`⤢`で一覧 |
| **一覧** | 全選択肢を番号・サブ説明つきで一覧。`⤡閉じる` | **行タップ＝即選択**／各行`詳細›`・行長押し＝詳細 |
| **詳細** | 1選択肢を深掘り。チップ/`‹ 前・次 ›`で巡回 | `これを選ぶ`＝確定／**`比較`で横並び**／`✕`＝戻る |
| **比較** | 全選択肢を**効き目（萌黄）/留意（黄土）**で見比べ | 各行`選ぶ`＝確定／話す可／`詳細`で深掘り |

- **行タップ＝即選択**（最速回答）。確認したい時だけ各行の `詳細 ›` で詳細シートを開く（決定 2026-06-25）。
- **近道（長押し）**: 最小構成の chip を**長押し**すると、一覧を経由せず**その選択肢の詳細**を直接開く（実演 Figma `167:2`）。
- **比較**: 詳細から「比較」へ切替で全選択肢を**効き目/留意**で横並び比較（Figma `165:2`）。
- **詳細の観点**: `どう動く？` / `効き目`（萌黄）/ `留意`（黄土）/ `関連・出所`（要件＋発話×画）。
- **回答確定はどの階層でも**：話す or chip/行/`これを選ぶ` → **選択肢UIは閉じて最小に戻る**（次の問いが来れば最小に新問い）。

### 選択肢の構成
- **動的 3〜4**（問いに応じて可変）＋ **常設 2**：
  - 「その他（話す / 入力で答える）」
  - 「わからない・あとで確認（保留）」
- 通常の問い＝**金枠**／矛盾検知＝**緋枠**（同一部品の色違い）。

## 5. ボトムバー（常時 2 行）& 入力系統

```
[ 🔈 消音 ]            [ 🎙 会話（マイク） ]   ← 1行目
[ テキストで入力…              ] [ ▶ ]      ← 2行目
```

- **2 系統の独立トグル**:
  - **会話（マイク）** = **音声入力 ON/OFF**（自分の声を拾うか）
  - **消音** = **音声出力 ON/OFF**（SANBA の読み上げを止め、字幕で進行）
- **音声 ⇄ テキストは併用**。テキスト送信でも phase は会話として進む（音声の代替/補助）。
- a11y: 見た目が古語でも `aria-label` は現代語機能名（例: マイク=「録音の開始/停止」、消音=「読み上げの消音」）。

## 6. 素材パイプライン（05 に内包）

```
05-2 手段選択（カメラ/アップロード/Drive）
  → アップロード（05-0 一覧に進捗%）
  → 背景解析（Gemini Vision / RAG grounding・領域検出/OCR/突合）  ← 会話は止めない
  → 05-1 詳細（抽出要件・言葉×画の矛盾）／06 要件絵巻へ反映
  → 矛盾は 04 の選択肢カード（緋）で起票
```

- 解析は**バックグラウンド**。進捗はミニ状況「📎資料 N（解析中…）」と 05 一覧のインラインバーで把握。

## 7. 判定（07）と結果（08）

- **判定の発火**: ①回答時・②ファイルアップロード時に**裏で確定ゲートを更新**、③**終了押下時**に判定画面で提示。
- **確定ゲート**: 未解消（矛盾/抜け/不明瞭）が **0 件**なら確定可、**1 件でも残れば確定不可**。
- **結果（08）**: 確定要件絵巻を**画面で確認＝必須**。**任意で** PDF 出力 / Google Drive 連携 / GitHub Issue 作成。

## 8. デザイントークン & アクセシビリティ

- **アートディレクション**: 産婆術アトリエ（Gilded Maieutics・金彩 serif）。フォント = Noto Serif JP。
- **色の意味写像**（[ubiquitous-language §0](ubiquitous-language.md) 準拠・**色のみに依存しない／ラベル＋アイコン併記**）:
  - 矛盾 = 緋 `#D2564B` ／ 抜け = 黄土 `#E0A93B` ／ 発話/解消 = 萌黄 ／ 確定/主操作 = 金（グラデ）
- **コピー = 全面古語**（演出）。ただし**機能名・契約・コードは機能名**（ubiquitous-language §13 対応表）。
- **a11y は現代語で担保**（ADR-0017）: `aria-label`・状態のラベル＋アイコン併記・WCAG AA。
