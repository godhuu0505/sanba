# 04 会話履歴（音声会話・選択肢・検知）

- **優先度**: **P0**（核となる対話・差別化）
- **Figma 正本**: 基準 `135:3`／状態: 待機 `145:3`・リスニング `145:80`・発話中(partial) `145:157`・読み上げ `145:234`・消音 `145:311`／
  選択肢: 一覧(最大化) `145:388`・詳細 `159:2`・比較 `165:2`・近道(長押し) `167:2`／終了確認 `148:188`／共通シェル土台 `127:2`
- **仕様**: [conversation-experience.md §3〜§5](../conversation-experience.md) / [ADR-0018](../../adr/0018-conversation-experience.md)

## 目的
音声で深掘りしながら、**3〜4 択で即答**でき、**矛盾/抜け**をその場で取り上げる。会話を止めずに資料・要件絵巻へも横断。
タブ共通シェルの「会話履歴」タブ。

## 常時 UI（全タブ共通・[§2](../conversation-experience.md)）
ヘッダー（産・問答・`● REC`・経過・終了⏹）／ミニ状況（◆要件 ⚠未確定 📎資料）／タブ／**問い＋選択肢ピン**／**2行ボトムバー**。

## 状態（会話履歴の本文）
| 状態 | 表示 |
|---|---|
| 待機 | オーブ脈動・「お話しください（待機）」 |
| リスニング | 「声を聞いています…」（萌黄）・マイク点灯 |
| 発話中(partial) | 認識途中の**薄字バブル**＋「‖ 認識中…」 |
| 思考中 | SANBA 思考インジケータ |
| 読み上げ中 | 「SANBA が申しております…」・マイク休止 |
| 消音 | 消音バナー＋ミュート点灯（出力停止・**字幕で進行**） |

## 選択肢（3 モード・[§4](../conversation-experience.md)）
- **最小**（常時ピン・横スクロールchip・`⤢広げる`）⇄ **一覧**（展開カード・行タップ=即選択・各行`詳細›`・`⤡閉じる`）⇄
  **詳細**（ボトムシート・チップ/`‹前 次›`巡回・`これを選ぶ`）。
  **比較**（効き目/留意で全選択肢を見比べ）は Figma 正本 `165:2` に存在するが§11で未決。実装スコープ確定前は任意扱い。
- 構成: 動的 3〜4 ＋ 常設 2（「その他＝話す/入力」「わからない・あとで＝保留」）。通常=金枠／**矛盾検知=緋枠**。
- 軸はグラバー「上=開く/下=閉じる」。回答はどの階層でも話す/タップで確定 → 最小に戻る。
- **近道**: 最小構成の chip を**長押し**すると、一覧を経由せず**その選択肢の詳細**を直接開く（`167:2`）。

## 検知（矛盾/抜け/不明瞭）
- 検知到着で選択肢カードが**緋**で割り込み（最小→必要なら展開）。種別: 矛盾=緋 `#D2564B` / 抜け=黄土 `#E0A93B` / 不明瞭。
- **色のみに依存せずラベル＋アイコン併記**。`refs`（根拠の発話）への導線を持つ。

## 入力コントロール（[§5](../conversation-experience.md)）
2 行ボトムバー：1行目 `[消音=出力ON/OFF][会話=マイク入力ON/OFF]`／2行目 `[テキスト入力][▶送信]`。音声⇄テキスト併用。

## 連携（[realtime-contract](../realtime-contract.md)）
- 購読: `detection.contradiction` / `detection.gap` / `status` / `transcript.partial` / `transcript.final`（字幕・発話中バブル・refs）。
- 回答: `user.selection` 送信 → agent がツール実行 → `detection.resolved` 受信でカードを閉じる。
- 復元: 途中参加/リロードは `GET /api/sessions/{id}/detections?open=1` で未解消を復元（ACK まで開いたまま）。
- 終了: ⏹ → 終了確認ダイアログ（`148:188`）。未解消 N 件を提示し 07 判定へ。

## 受け入れ基準（AC）
- [ ] 待機/リスニング/発話中(partial)/読み上げ/消音 が表示・操作で判別できる。
- [ ] 選択肢が**最小⇄一覧⇄詳細⇄比較**で切替でき、行タップ=即選択・各行`詳細›`で詳細が開く。
- [ ] 最小構成の chip **長押し**で詳細を直接開ける（一覧を経由しない近道）。
- [ ] 矛盾/抜けが**ラベルとアイコン**で判別でき、色覚に依存しない。`refs` をたどれる。
- [ ] 複数検知が seq で時系列に積まれ、`(type,id)` で重複しない。
- [ ] 消音中は出力停止でも**字幕で会話が追える**。マイク（会話）と消音が独立に効く。
- [ ] テキスト入力でも回答でき、音声と併用できる。古語表示でも a11y は現代語。
